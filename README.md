# Sticky Notes

A simple and responsive Sticky Notes web application built using **HTML, CSS, and JavaScript**. This project allows users to create, view, and delete notes directly in the browser, providing a lightweight solution for managing quick reminders and ideas.

## Features

* Create new notes instantly
* Display notes in a clean card-based layout
* Delete notes with a single click
* Responsive and modern user interface
* Lightweight and easy to use

## How It Works

1. Click the **Create Notes** button.
2. A new note card is generated.
3. Enter your text inside the note.
4. Click the **🗑️ Delete** icon to remove a note.
